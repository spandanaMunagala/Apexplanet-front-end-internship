Basics of Web Development

HTML • CSS • JavaScript

HTML

HTML (HyperText Markup Language) is used to create the structure of web pages using elements like headings, paragraphs, images, and links.

CSS

CSS (Cascading Style Sheets) is used to style the webpage by adding colors, fonts, spacing, and layouts.

JavaScript

JavaScript adds interactivity to web pages such as button clicks, alerts, form validation, and dynamic content updates.
